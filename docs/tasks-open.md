## Безопасность ПД (152-ФЗ) — сквозное
- [ ] 🔴 **Маскирование ПД перед отправкой в Claude:** в LLM уходит **текст диалога с вырезанными сильными идентификаторами** (телефон/фамилия/адрес) + короткий обезличенный ярлык-шапка («лид #412, мама Гузель, ребёнок Тимур 2.5 г»). Имена для общения **оставляем** — бот живой. **НЕ** сводим переписку к сухой анкете (это убьёт живой диалог). Опознание «кто это» делает наш код, не LLM. *(Актуально с F1.)*
- [ ] 🔴 **Anthropic недоступен в РФ → Claude через прокси/релей в разрешённой стране.** Хранилище ПД (Redis/транскрипты) и маппинг `lead_id↔ПД` — **в РФ**; вызов LLM уходит за рубеж = трансграничная передача (под согласие). Прямой вызов с РФ-IP, вероятно, блокируется. Сетевой путь решаем на деплое; дизайн F1 не меняет.
- [ ] 🔴 **Согласие — взято в работу (заказчик):** трансграничная передача в Claude/США + субъекты-несовершеннолетние. Остаётся: **сервер в РФ** — направление **RF VPS** (Selectel / Timeweb Cloud, 152-ФЗ-совместимый), финализировать до хранения транскриптов.
- ⏸️ *Отложено:* шифрование склада (Redis/бэкапы) — не сейчас, опционально к проду.
