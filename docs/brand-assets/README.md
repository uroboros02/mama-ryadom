# Визуал бренда «Мама рядом»

Фирменный стиль от дизайнера **@diana_alexseevna** (получен 2026-06-21). Описание визуала —
в [`../brand-platform.md`](../brand-platform.md) §6 (палитра, шрифты, лого, тон).

## Что в git (ядро айдентики)
Лежит в `Мама рядом Бренд/`:
- **Логотип/** — цвет + Ч/Б (рука мамы с детской ладошкой), `.png`.
- **Цветовое решение/** — палитра с кодами (`#7393CC`, `#FFF7B0`, `#FFE5D1`, `#546A88`, `#CFE1FF`).
- **Шрифты/** — Neucha (акцентный) + Nunito Sans (основной), `.ttf`. Оба — бесплатные Google Fonts.
- **Элементы/** — клякса, ромашка, сердце, контур-кружок (`.svg` + `.png`).

## Что НЕ в git (тяжёлое, ~22 МБ — см. `.gitignore`)
Лежит на диске локально, но не коммитится: **брендбук-PDF** (15 стр.), мокапы мерча, визитки,
бланки, карта партнёра, аватарки, фоны/паттерны. Полный комплект — у клиента и дизайнера.
Вся нужная инфа из брендбука уже перенесена в `brand-platform.md` §6.
