# Бренд-платформа в PDF (§7)

Готовый документ для людей (сотрудники, партнёры, подрядчики) — вся платформа простым языком,
в брендовых цветах и шрифтах.

- **Готовый файл:** `Бренд-платформа Мама рядом.pdf` (8 стр.)
- **Источник содержания:** [`../brand-platform.md`](../brand-platform.md) (§1–§6). В PDF идёт
  «чистая» версия текста — без внутренней кухни (ссылок «§», термина «onlyness», архивов вариантов).

## Как пересобрать
1. Сгенерировать HTML (встраивает шрифты Neucha + Nunito Sans и логотип из `../brand-assets/` как base64):

       python3 build_pdf.py

   Создаёт `brand-platform.html` (промежуточный артефакт, в git не хранится).

2. Отрендерить HTML в PDF через Chrome (без новых зависимостей):

       "/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" \
         --headless --disable-gpu --no-pdf-header-footer \
         --print-to-pdf="Бренд-платформа Мама рядом.pdf" brand-platform.html

## Правка текста
Менять текст/вёрстку — в `build_pdf.py` (содержание секций и CSS прямо в нём), потом пересобрать.
`build_pdf.py` использует **только стандартную библиотеку Python** — ставить ничего не нужно.

## Что в git
- `build_pdf.py` — генератор (источник, регенерируемо).
- `Бренд-платформа Мама рядом.pdf` — готовый документ.
- `brand-platform.html` — **не в git** (генерируемый артефакт ~1 МБ, см. `.gitignore`).
